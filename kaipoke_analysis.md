# カイポケ（Kaipoke）分析レポート

## 基本情報

- **提供会社**: SMS株式会社
- **サービス形態**: クラウド型SaaS
- **対象**: 介護・看護・障害福祉事業所
- **料金**: 初期費用0円、3年間無料プランあり

## 主要機能

### 1. ソフトの機能
- 利用者情報管理
- 介護記録・看護記録の作成
- タブレット・スマホでの記録入力
- 介護計画書・ケアプラン作成
- 国保連請求（伝送機能）
- 利用者負担金の請求
- 各種帳票作成

### 2. 対応サービス
- 居宅介護支援
- 通所介護
- 訪問介護（介護・障害福祉）
- 福祉用具貸与・販売
- その他多数のサービス種別

### 3. 開業支援サービス
- 開業準備から日々の業務・経営改善まで事業所運営をトータルサポート

### 4. 経営支援サービス
- 採用支援
- 営業支援
- 口座振替サービス
- スマホレンタル

## カイポケの強み

1. **一気通貫の業務フロー**: 記録→実績→請求がシームレスに連動
2. **タブレット・スマホ対応**: 現場での記録入力が可能
3. **低価格**: 3年間無料、初期費用0円
4. **シェア率が高い**: 業界で広く使われている
5. **ライセンスフリー**: 複数端末での利用が可能
6. **経営支援サービス**: 採用・営業などの支援も提供

## カイポケの弱み（推測）

1. **AI機能の欠如**: 記録の質評価、予測機能などのAI連携がない
2. **記録作成の負担**: 職員が一から記録を作成する必要がある
3. **カスタマイズ性**: クラウド型のため、事業所ごとのカスタマイズが限定的
4. **連携機能**: 他システムとの連携が最低限に抑えられている
5. **データ分析**: 高度なデータ分析や予測機能が不足
